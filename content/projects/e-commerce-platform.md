A full-stack e-commerce platform built from the ground up. Features real-time inventory management, Stripe payment processing, and a responsive storefront designed for conversion.

The backend leverages Supabase for authentication, database, and real-time subscriptions, while the frontend is built with Next.js and Tailwind CSS for a fast, accessible shopping experience.
