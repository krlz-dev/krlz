An interactive analytics dashboard that transforms raw data into actionable insights. Features custom D3.js visualizations, real-time metric updates, and a role-based access control system.

Built with a React frontend consuming a Node.js API backed by PostgreSQL, with WebSocket connections for live data streaming.
