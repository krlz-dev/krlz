Co-founded with [maryaleks.dev](https://maryaleks.dev). kit-a is a browser-based application for creating system architecture diagrams and project timelines — no installation required.

The architecture tool offers 14 component types (Client, Server, DB, API, Auth, Queue, Cache, Cloud, Function, and more), smart connections with labels, colors, and animated flow visualization, plus six starter templates including 3-Tier Web App, Microservices, and Event-Driven patterns.

The integrated Gantt planner supports six view modes, task dependencies, milestones, progress tracking, drag-to-reschedule, and team member assignment with color coding.

Export options include PNG (2x resolution), JPG, animated GIF, and WebM video. Projects can be saved to the cloud or imported/exported as JSON.
