As a Senior Developer at Portavita Health (2023-2024) in Innopolis, Russia, I led the implementation of critical healthcare software, collaborating with the Tatarstan government to introduce new systems and modernize legacy infrastructure.

Led the evaluation and selection of backend systems, successfully adopting FHIR HAPI and Spring Boot frameworks to improve system efficiency and interoperability. Built Python-based tools to streamline FHIR resource generation, accelerating the development process and enabling timely MVP delivery.

Implemented backend integration with legacy systems using Kafka for rapid data migration while maintaining operational continuity. Oversaw frontend development with React and Next.js, including OpenID registration flows. Designed and implemented medical examination processes in HAPI, creating intuitive interfaces to support flexible treatment options.
